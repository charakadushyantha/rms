================================================================================
  SETUP IN 3 STEPS (5 MINUTES)
================================================================================

STEP 1: CREATE DATABASE TABLES
-------------------------------
Run this command in your terminal:

    mysql -u root -p rmsdb < Database/chatbot_schema.sql

Or open phpMyAdmin and run the SQL from: Database/chatbot_schema.sql


STEP 2: GET & ADD API KEY
--------------------------
1. Go to: https://platform.openai.com/api-keys
2. Sign up/login (free account works)
3. Click "Create new secret key"
4. Copy the key (starts with sk-...)
5. Edit: application/config/chatbot.php
6. Replace 'YOUR_API_KEY_HERE' with your actual key


STEP 3: ADD TO YOUR PAGES
--------------------------
Add this line before </body> tag in any page:

    <?php $this->load->view('chatbot_widget'); ?>

Example pages to add it:
- application/views/login.php
- application/views/signup.php
- application/views/Admin_dashboard_view/admin_dashboard.php


STEP 4: TEST IT!
----------------
Visit: http://localhost/rms/chatbot_test

Look for the blue chat button in bottom-right corner.
Click it and type: "What job openings do you have?"

================================================================================
  CUSTOMIZATION
================================================================================

Change Colors/Position:
  Edit: application/config/chatbot.php
  
  $config['widget_color'] = '#007bff';      // Your brand color
  $config['widget_position'] = 'bottom-right';  // or 'bottom-left'
  $config['widget_title'] = 'Recruitment Assistant';

Customize AI Responses:
  Edit: application/controllers/Chatbot.php
  Find: _get_system_prompt() function
  Add your company info, job openings, benefits, etc.

================================================================================
  TROUBLESHOOTING
================================================================================

Chat button not showing?
  → Check browser console (F12) for errors
  → Verify Assets/js/chatbot.js exists
  → Clear browser cache

"API key not configured" error?
  → Check application/config/chatbot.php
  → Make sure key starts with sk-
  → Verify key at https://platform.openai.com

Database errors?
  → Run Database/chatbot_schema.sql
  → Check database connection in application/config/database.php
  → Verify database name is 'rmsdb'
