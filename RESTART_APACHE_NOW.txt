═══════════════════════════════════════════════════════════════
  🎯 WHY THIS HAPPENS
═══════════════════════════════════════════════════════════════

PHP caches compiled code for performance (OPcache).
When you update a PHP file, Apache might still use the old cached
version until you restart it.

This is normal and happens with all PHP applications.
