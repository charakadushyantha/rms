🤖 Chatbot Test Page

Look for the blue chat button in the bottom-right corner!

If you see a blue circular button with a chat icon, the chatbot is working!

Click it to open the chat window.

Troubleshooting:

  1. Open browser console (F12) and check for errors
  2. Make sure Assets/js/chatbot.js exists
  3. Check if the script is loading (Network tab in F12)
