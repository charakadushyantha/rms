📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Page shows "Cannot redeclare" error
Solution: Clear browser cache, restart Apache

Problem: Upload fails with "Unknown column" error
Solution: Verify database has correct columns (see DOCUMENT_UPLOAD_FIXED.md)

Problem: View button doesn't work
Solution: Check if file exists in uploads/candidate_documents/

Problem: Download button doesn't work
Solution: Check file permissions on uploads folder

Problem: Delete button doesn't work
Solution: Check browser console for JavaScript errors
