💡 WHAT CHANGED
═══════════════════════════════════════════════════════════════════════

BEFORE (Old Way):
─────────────────────────────────────────────────────────────────────
Every script had:
  $host = 'localhost';
  $username = 'root';
  $password = '';
  $database = 'rmsdb';
  $conn = new mysqli($host, $username, $password, $database);

Problem: Need to update 50+ files to change credentials!

AFTER (New Way):
─────────────────────────────────────────────────────────────────────
Scripts now have:
  require_once __DIR__ . '/config/database.php';
  $conn = getDatabaseConnection();

Solution: Change once in config/database.php, applies everywhere!
