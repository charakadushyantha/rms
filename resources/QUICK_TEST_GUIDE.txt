┌───────────────────────────────────────────────────────────────┐
│ STEP 1: CLEAR CACHE                                           │
└───────────────────────────────────────────────────────────────┘
Press: Ctrl + Shift + R (or Ctrl + F5)

┌───────────────────────────────────────────────────────────────┐
│ STEP 2: OPEN DOCUMENTS PAGE                                   │
└───────────────────────────────────────────────────────────────┘
URL: http://localhost/rms/C_dashboard/documents

Expected: Page loads without errors ✅

┌───────────────────────────────────────────────────────────────┐
│ STEP 3: UPLOAD A FILE                                         │
└───────────────────────────────────────────────────────────────┘
1. Select document type: "Resume"
2. Choose a PDF file
3. Click "Upload Document"

Expected: Success message + page reloads ✅

┌───────────────────────────────────────────────────────────────┐
│ STEP 4: VIEW THE DOCUMENT                                     │
└───────────────────────────────────────────────────────────────┘
Click the BLUE EYE ICON (👁️)

Expected: Opens in new tab ✅

┌───────────────────────────────────────────────────────────────┐
│ STEP 5: DOWNLOAD THE DOCUMENT                                 │
└───────────────────────────────────────────────────────────────┘
Click the BLUE DOWNLOAD ICON (⬇️)

Expected: File downloads to your computer ✅

┌───────────────────────────────────────────────────────────────┐
│ STEP 6: DELETE THE DOCUMENT                                   │
└───────────────────────────────────────────────────────────────┘
Click the RED TRASH ICON (🗑️)
Click "Yes, delete it" in confirmation

Expected: Document removed from list ✅

👁️  VIEW BUTTON (Blue Eye Icon)
   → Opens document in new browser tab
   → PDF and images display inline
   → Word docs may download

⬇️  DOWNLOAD BUTTON (Blue Download Icon)
   → Forces file download
   → Saves to Downloads folder
   → Works for all file types

🗑️  DELETE BUTTON (Red Trash Icon)
   → Shows confirmation modal
   → Deletes file from server
   → Removes from database
