┌──────────────────────────────────────────────────────────────────────────────┐
│ ❓ TROUBLESHOOTING                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

  Still seeing blank page?
  ─────────────────────────────────────────────────────────────────────────
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Check browser console for errors (F12)
  3. Verify database tables exist
  4. Check Apache error logs


  Bot not responding?
  ─────────────────────────────────────────────────────────────────────────
  1. Check database connection in config/database.php
  2. Verify Bot_model.php exists
  3. Check BotEngine.php library
  4. Look for AJAX errors in browser console


  Upload not working?
  ─────────────────────────────────────────────────────────────────────────
  1. Check uploads/cvs/ folder exists
  2. Verify folder permissions (755)
  3. Check PHP upload_max_filesize setting
  4. Verify file format (PDF, DOC, DOCX, TXT)
