  Common Issues:
  ─────────────────────────────────────────────────────────────────────────
  
  Database Connection Error:
  → Check config/database.php
  → Verify MySQL is running
  
  Bot Not Responding:
  → Run create_bot_tables.php
  → Check browser console
  
  Interview Link Not Working:
  → Verify token is correct
  → Check expiration date
